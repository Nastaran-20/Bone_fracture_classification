#Bone Fracture Classification
A CNN model built from scratch to classify X-ray images into fractured and not fractured bones using PyTorch.

Features
Custom CNN with BatchNorm & Dropout
Data augmentation and weighted loss for class imbalance
Evaluated with accuracy, confusion matrix, and classification report
Results
Test Accuracy: 96.6%
Very few misclassifications

Usage
git clone https://github.com/YOUR_USERNAME/Bone-Fracture-Classification.git
pip install torch torchvision matplotlib scikit-learn pillow

Run the notebook: BoneFracture.ipynb
